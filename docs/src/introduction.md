# Introduction

SemASM is multi-architecture semantic infrastructure for software written directly in assembly. It provides portable contracts, target kits, ASIR-based analysis, and verification pipelines. It does not replace assemblers or claim that assembly always beats optimizing compilers.

**Current status:** early stabilization. The repository contains partial target
implementations, but no target is currently CI-verified or release-qualified.
Read the root `README.md` and `capabilities.toml` for evidence levels and known
limitations.
